# Platform Engineering Starter Kit

## Summary
Built a platform “starter kit” that standardizes service delivery: deploy, observe, and operate.

## Golden Path
- Service scaffold
- CI pipeline template
- Kubernetes deployment standard
- Observability + alerting baseline
- Documentation and onboarding checklist

## Results / Impact
- Reduced onboarding time for new services
- Consistent deployment + monitoring patterns
- Less tribal knowledge, more self-service

## Tech
Kubernetes (GKE), GitOps (Flux), GitLab CI, Terraform, Observability tooling
